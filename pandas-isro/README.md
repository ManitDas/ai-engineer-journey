# ISRO Satellite Data Analysis

Exploratory data analysis of ISRO's satellite launch history using Pandas and Matplotlib.

## Dataset
ISRO Satellite Complete Dataset from Kaggle — 40 satellites, 30 features.

## Analysis
- Satellite distribution by purpose
- Launch trend over time (2003–2020)
- Most used launch vehicles
- Orbit class distribution
- Launch mass distribution
- Average mass by satellite purpose

## Key Findings
- Earth Observation is ISRO's primary focus (18 satellites)
- PSLV is the most used rocket (14 launches)
- Launch activity peaked in 2016
- Communications satellites are heaviest on average

## Tools
Python, Pandas, Matplotlib, Seaborn, Jupyter
